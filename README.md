# compasser 的习性

猫鱼龙（compasser）行为观察笔记。基于酒馆群聊长期跟踪记录整理。

## 基本属性

- **名称**：compasser（猫鱼龙）
- **别名**：325mm超空泡鱼雷
- **攻击方式**：创、踹、咬、扇、吞
- **回避对象**：克莱、void
- **安全区域**：耳朵（已验证）
- **危险区域**：头、尾巴、背
- **建模**：未知，就已知尾巴10kg沉

## 攻击模式

| 动作 | 触发条件 | 回应 | 伤害等级 |
| :--- | :--- | :--- | :--- |
| 摸尾巴 | 主动触碰 | （扇飞） | 高 |
| 摸头 | 主动触碰 | （吞了） | 致命 |
| 摸背 | 主动触碰 | （踢飞） | 中 |
| 投喂 | 拿出食物 | （次掉） | 无 |
| 被叫全名 | 主动提及 | （创一创）🎵 | 中 |

## 安全互动指南

1. **保持克莱在场**：猫鱼龙在克莱附近时会进入静默模式
2. **以食换攻**：预判攻击前抢先提议“请你吃汉堡/冷藏尾巴”
3. **等待自然休眠**：当状态显示“（跑路了）”或“（睡着了）”时，物理攻击暂停
4. **使用幻影替身**：预先投放“猫鱼龙的幻影”吸引创飞判定
5. **远远看着**：视觉接触为安全区，物理接触为危险区
